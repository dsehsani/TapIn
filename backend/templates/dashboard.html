What counts as a DAU?

A user is counted as a Daily Active User (DAU) once per day when they perform at least one meaningful engagement action. Each action is deduplicated per user per day — a user active 10 times still counts as 1 DAU.

📰 Article Read User opens an article and spends at least 30 seconds reading. Tapping in alone does not count.
📅 Event Engaged User saves an event or taps "View on Aggie Life" — clear signals of genuine interest.
🟩 Wordle Played User starts a Wordle game session for the day.
🔊 Echo Played User starts an Echo memory game session.
🔧 Pipes Played User starts a Pipes puzzle session.
